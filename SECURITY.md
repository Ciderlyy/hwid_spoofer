# Security Policy

## Supported Versions

This project is for **security research and red-teaming** of your own systems only. It is not intended for production use.

## Reporting a Vulnerability

If you discover a vulnerability in this software that could affect users running it in their research environments:

1. **Do not** open a public issue.
2. Open a private security advisory on GitHub, or contact the maintainer directly if you have their contact info.
3. Provide steps to reproduce and impact assessment.

We will respond as soon as possible. There are no bug bounties.

## Responsible Use

- Use only in isolated VMs (e.g., VMware with snapshots).
- Do not use to evade anti-cheat or licensing on systems you do not own.
- The authors assume no liability for misuse.
